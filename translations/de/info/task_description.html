Einer der Roboter hat eine einfache Aufgabe bekommen: Er soll aus einer Folge einzelner Wörter einen Satz bilden, der dann als Wegbeschreibung dienen soll, die um das Raumschiff herum führt. Allerdings ist dieser Roboter Linkshänder und obendrein ein Spaßvogel. Er kann es einfach nicht lassen, seine rechtshändigen Kollegen zu veräppeln.

Gegeben ist eine Folge von einzelnen Worten als Strings. Du sollst aus diesen einzelnen Worten einen zusammenhängenden String bilden, in welchem die einzelnen Worte durch Kommata getrennt sind. Um die rechtshändigen Roboter zu veräppeln, sollst Du außerdem jedes "right" durch "left" ersetzen, auch wenn das "right" nur Teil eines Wortes ist. Alle Worte sind klein geschrieben!

Eingabe: Eine Folge von Worten als Tuple (unicode).

Ausgabe: Den Text als String (einzelne Worte durch Kommata getrennt).
