Uno dei robot è incaricato di un semplice compito: unire una sequenza di stringhe un una frase per produrre istruzioni su come spostarsi sulla nave. Tuttavia questo robot è mancino ed ha la tendenza a burlarsi dei suoi amici destrorsi confondendoli.

Ti viene data una sequenza di stringhe. Devi unire queste stringhe un un blocco di testo in cui le stringhe iniziali siano separate da vigole. Per fare uno scherzo ai robot destrorsi devi rimpiazzare la parola "right" con la parola "left" ogni volta che appare, anche se è parte di un'altra parola. Tutte le stringhe sono date in caratteri minuscoli.

Input: Una sequenza di stringhe come tupla di stringhe (unicode).

Output: Il testo come una stringa.

Esempio: 

left_join(("left", "right", "left", "stop")) == "left,left,left,stop"
left_join(("bright aright", "ok")) == "bleft aleft,ok"
left_join(("brightness wright",)) == "bleftness wleft"
left_join(("enough", "jokes")) == "enough,jokes"

Come si usa: Questo è un semplice esempio di operazioni effettuate con stringhe e sequenze.

Prerequisiti:
0 < len(phrases) < 42
