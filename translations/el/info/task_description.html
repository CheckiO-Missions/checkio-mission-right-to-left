Ένα από τα ρομπότ αναλαμβάνει μια απλή εργασία: να ενώσει μια σειρά από συμβολοσειρές σε μια πρόταση, ώστε να παράγει οδηγίες για το πώς θα μετακινηθει κανείς μέσα στο διαστημόπλοιο. Αλλά το ρομπότ είναι αριστερόχειρας και έχει την τάση να κάνει πλάκα και να μπερδεύει τους δεξιόχειρες φίλους του.

Σου δίνεται μια σειρά από strings. Αποστολή σου είναι να ενώσεις αυτά τα strings σε ένα ενιαίο κείμενο, μιας και τα αρχικά είναι χωρισμένα μεταξύ τους με κόμμα. Για να κάνεις πλάκα στα δεξιόχειρα ρομπότ πρέπει να αντικαταστήσεις όλες τις εμφανίσεις της λέξης "δεξιά" με τη λέξη "αριστερά", ακόμα και αν αποτελεί μέρος μιας άλλης λέξης. Όλες οι λέξεις δίνονται με πεζά γράμματα.

Είσοδος: Μια ακολουθία από strings με τη μορφή πλειάδας (unicode).

Έξοδος: Το κείμενο σε μορφή string.

Παράδειγμα: 

left_join(("left", "right", "left", "stop")) == "left,left,left,stop"
left_join(("bright aright", "ok")) == "bleft aleft,ok"
left_join(("brightness wright",)) == "bleftness wleft"
left_join(("enough", "jokes")) == "enough,jokes"

Πώς χρησιμοποιείται: Ένα απλό παράδειγμα επεξεργασίας συμβολοσειρών.
