Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Może Pan zastąpić podciągi metodą replace() 

"abcd ababba".replace("ab", "0") == "0cd 00ba"

Potrzebuję pomocy, aby kontynuować misję.

Mogą Państwo dołączyć do ciągów za pomocą metody join(). 

":".join(["A", "1", "cd"]) == "A:1:cd"

Jestem w połowie drogi. Proszę o pomoc!

Proszę połączyć wszystkie podane ciągi, następnie zamienić "prawy" na "lewy" i zwrócić (nie drukować) wynik. 

def left_join(phrases: tuple[str]) -> str:

    join_phrase = ','.join(phrases)
    result = join_phrase.replace('right','left')

    return result
